## Getting Started
The easiest way to have the repo compiled is typing
<pre>
$ git clone git@github.com:mcarosio80/AoC2023.git
$ cd AoC2023
$ mkdir -p build/
$ cd build/
$ cmake ..
$ cmake --build . -j24
</pre>

Have fun and long life to [AoC](https://adventofcode.com/2023)!
